# Roadmap

## v0.1 — Single-player playable (this milestone)
- Menu, single-player setup, in-game, game-over screens
- 1–6 cars in arena with full physics
- Spatial engine sounds, proximity beeps, collision/wall sounds
- AI bots that pursue and brawl
- Q-key targeting sweep, ARIA live announcements
- Score + personal-best persistence
- Gamepad + haptics

## v0.2 — Polish
- Per-car distinct engine timbres tuned by ear
- Optional speech-synthesis announcer
- Settings screen (volume, haptics, TTS toggle)
- Tutorial mode (guided practice with AI 1)

## v0.3 — Multiplayer (stub → real)
- WebRTC datachannel transport
- Lobby (host/join via short code)
- Lockstep input frames + periodic state correction
- Spectator mode after elimination

## v0.4 — Variety
- Round mode (best of N)
- Power-ups (speed boost, repair, sonar ping)
- Larger / shaped arenas
